Flagship Project: MathFractal Engine

The story behind our numpy-optimized ASCII fractal engine rendering complex equations.

Told as a Team Story

It started during a whiteboard session in Lab Komputer. We wanted a lightweight tool to generate and output Mandelbrot fractal structures in the console without rendering heavy window elements.

The Collaboration: Sarah worked out the mathematical matrix equations, while Alex wrote the NumPy array broadcasting configurations to speed up math executions. Chris styled the command parameters parser.

We hit performance limits when iterating arrays past 100 loops. By mapping the math loops inside memory slices, we optimized it to compile coordinates in less than 8ms. Today, MathFractal Engine displays stunning fractals right in any basic terminal window!
